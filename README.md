## 2.How to run the app

1.Clone the develop branch.
> $git clone https://github.com/NataliaKirik/Calculator.git

2.Go to the directory
> $ cd Calculator

3.Install the npm modules
> $ npm install

4.Run the app
> $ npm run dev / npm run build
